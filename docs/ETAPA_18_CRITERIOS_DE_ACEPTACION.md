# ETAPA 18 — Criterios de aceptación

- [x] Existe una plantilla v1 completamente ficticia y sin datos personales.
- [x] El paquete usa claves externas y no permite imponer IDs internos.
- [x] Solo un administrador activo puede ejecutar la Callable en el emulador.
- [x] Estructura, tamaño, cantidades, relaciones, ciclos, códigos y secretos se validan sin escribir.
- [x] El total se calcula centralmente y total cero permanece bloqueado.
- [x] Catálogo, inventario y ocupaciones actuales se comparan y resumen.
- [x] El hash SHA-256 es determinista ante orden no significativo.
- [x] Firestore queda idéntico antes y después, incluidas auditoría e idempotencia.
- [x] Maestro selecciona el archivo solo en memoria, filtra hallazgos y exporta un informe seguro.
- [x] La interfaz declara que no se realizó ninguna importación.
- [x] Campo no recibe cambios funcionales.
- [x] No existe Firebase productivo, despliegue, APK ni instalador.

`aptoParaImportar` no constituye autorización. Importación, reversión, migración real, datos reales y asignación definitiva de IDs permanecen fuera de alcance.
